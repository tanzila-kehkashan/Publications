### 3. UPD: A Plagiarism Detection Tool for Urdu Language Documents (2018)

[![ISSN](https://img.shields.io/badge/ISSN-2045--7057-blue)](http://www.ijmse.org/)

**Journal:** International Journal of Multidisciplinary Sciences and Engineering
**Year:** 2018 | **Volume:** 9, Issue 1 | **Pages:** 19-22

**Key Achievements:**
- First plagiarism detection tool for Urdu language
- Tokenization, trigram chunking, and absolute hashing
- Statistically validated using T-test
- Handles varying similarity levels (0% to 100%)

**Technologies:** Python, NLP, Urdu Text Processing, Statistical Analysis

## ğŸš€ Quick Start

### Clone Repository

```bash
git clone https://github.com/tanzila-kehkashan/Publications.git
cd Publications
```

### Install Dependencies (Example for Wheat Disease Detection)

```bash
cd "Journal Articles/Paper-3-Wheat-Disease-Detection/code"
pip install -r requirements.txt
```

### Run Training

```bash
python train_swin_transformer.py --epochs 10 --batch-size 32
```

### Run Inference

```bash
python inference.py --image path/to/wheat_leaf.jpg --model checkpoints/best_model.pth
```
